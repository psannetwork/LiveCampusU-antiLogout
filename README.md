# LiveCampusU-antiLogout

LiveCampusU（静岡大学学務システム）の自動ログアウトを防止するためのツールです。

## 概要
LiveCampusUは一定時間操作がないと自動的にログアウトされてしまいますが、このスクリプトはバックグラウンドで定期的にサーバーへリクエストを送信することで、セッションを維持（キープアライブ）します。

> [!WARNING]
> 本ツールの使用は自己責任でお願いします。サーバーへの過度な負荷を避けるため、実行間隔の設定には注意してください。

## 特徴
*   **GAS (Google Apps Script) 対応**: サーバーレスで24時間稼働が可能です。
*   **軽量動作**: 特定のAPIエンドポイントを叩くだけの最小限の通信。
*   **異常検知**: ログアウトを検知した際のログ出力機能。

## 使い方

### 1. GASプロジェクトの作成
1. [Google Apps Script](https://script.google.com/) にアクセスし、新しいプロジェクトを作成します。
2. `コード.gs` に本リポジトリのコードを貼り付けます。

### 2. Cookieの取得
1. ブラウザでLiveCampusUにログインします。
2. デベロッパーツール（F12）を開き、「ネットワーク」タブを確認します。
3. 適当な通信（スケジュール表示など）を選択し、リクエストヘッダーから `Cookie` の値をコピーします。
4. スクリプト内の `myCookie` 変数に貼り付けます。

### 3. トリガーの設定
1. GASエディタ左側の「トリガー（時計アイコン）」をクリックします。
2. 「トリガーを追加」を選択。
3. `keepMyGakujoAlive` を「時間手動型」で、**10分〜20分おき**に実行するように設定します。
